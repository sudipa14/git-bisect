Git bisect

How does git bisect works?

Key points about git bisect

  • We need to define whether a commit is good or bad
  • To start bisect session we need to write "git bisect start"
  • Define current status as good or bad
  • Define a good status also using git hash

Some of the important git bisect commands

  • Git bisect start
  • Git bisect bad
  • Git bisect good
  • Git bisect help
  • Git bisect reset
